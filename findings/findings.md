# Findings

## Event ID 4625 - Failed Login

Observation:
Multiple failed login attempts were detected.

Impact:
Repeated failed logins may indicate password guessing or brute-force attempts.

---

## Event ID 4624 - Successful Login

Observation:
A successful authentication event was recorded after login.

Impact:
Confirms successful user access to the system.

---

## Conclusion

Windows Security Logs were analyzed successfully. Both failed and successful authentication events were identified and investigated.
