
# Security Log Analysis Project

## Objective

Analyze Windows Security Logs to identify authentication events and potential security issues.

## Tools Used

* Windows Event Viewer
* Windows Security Logs

## Event IDs Investigated

* Event ID 4624 (Successful Login)
* Event ID 4625 (Failed Login)

## Activities Performed

* Generated failed login attempts in a lab environment
* Analyzed Windows Security Logs
* Investigated authentication events
* Documented findings and recommendations

## Key Findings

* Multiple failed login attempts were detected
* Successful authentication events were verified
* Security monitoring recommendations were documented

## Skills Demonstrated

* Log Analysis
* Security Monitoring
* Event Investigation
* Windows Security Logs
* SOC Analyst Fundamentals
