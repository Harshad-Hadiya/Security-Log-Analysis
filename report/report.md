# Security Log Analysis Project

## Objective

Analyze Windows Security Logs to identify suspicious authentication activities.

## Tools Used

* Windows Event Viewer
* Windows Security Logs

## Investigated Event IDs

* 4624 (Successful Login)
* 4625 (Failed Login)

## Analysis

Failed login attempts were generated in a controlled lab environment and identified using Event Viewer.

Successful authentication events were also reviewed to verify login activity.

## Security Recommendations

* Enable Account Lockout Policy
* Use Strong Passwords
* Monitor Failed Login Attempts
* Review Authentication Logs Regularly

## Outcome

Successfully identified and analyzed Windows authentication events and documented findings.
